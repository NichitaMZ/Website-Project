# Clothing Website Project

## Description
This project is a simple e-commerce clothing website built using HTML and CSS. It includes multiple pages such as Home, Collections, Arrivals, About, and Contact.

## Features
- Multi-page website
- Clean layout and navigation
- Styled using CSS

## Technologies Used
- HTML
- CSS

## How to Run
Download the files and open `index.html` in your browser.
